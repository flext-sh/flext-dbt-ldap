- **LDAP Directory Analytics**: Transformations for LDAP/Active Directory data
- **Dimensional Modeling**: Star schema design for user, group, and organizational analytics
- **LDAP-Specific Macros**: Custom dbt macros for DN processing and attribute handling
- **Historical Tracking**: Snapshot models for tracking LDAP entry changes over time
- **Clean Architecture**: Python components with domain-driven design patterns
